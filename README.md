# 五子棋游戏 (Gomoku)

一个基于 C++ 实现的命令行五子棋游戏。

## 项目简介

本项目是一个简单的五子棋游戏实现，支持双人对战。游戏在命令行界面运行，玩家通过输入坐标来落子，先连成五子者获胜。

## 功能特性

- **双人对战**：支持黑方和白方轮流落子
- **胜负判定**：自动检测五子连珠
- **坐标输入**：通过标准输入进行落子
- **棋盘显示**：使用 Unicode 字符显示棋盘状态
  - `●` 表示黑子
  - `○` 表示白子
  - ` ` 表示空位

## 编译运行

### 环境要求

- C++11 或更高版本编译器
- Windows/Linux/macOS

### 编译命令

```bash
g++ -std=c++11 -o gomoku main.cpp
```

### 运行游戏

```bash
./gomoku
```

## 游戏说明

1. 游戏开始后，黑方先行
2. 玩家输入坐标 `x y` 进行落子（坐标范围：1-38）
3. 系统会自动判断胜负和平局
4. 任意一方连成五子即获胜
5. 棋盘填满未分胜负则为平局

## 技术细节

### 类设计

- `GoGame`: 游戏核心类，包含棋盘状态和游戏逻辑
  - `set(x, y, player)`: 落子
  - `validPosition(x, y)`: 检查坐标合法性
  - `Win(x, y, player)`: 检查是否获胜
  - `GameOver()`: 检查游戏是否结束
  - `show()`: 显示棋盘

### 游戏规则

- 棋盘大小：38×38
- 胜利条件：任意方向（横、竖、斜）连成五子
- 坐标系统：1-based 索引
